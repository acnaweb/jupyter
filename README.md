# Notebooks para Data Science

Organização e EDA com Jupyter Notebooks

### Versioning

- Proposta de nomenclatura de arquivos de notebook.
```
[ISO 8601 date]-[DS-initials]-[2-4 word description].ipynb
```

- Exemplo
```
2023-03-26-ac-eda.ipynb
```
